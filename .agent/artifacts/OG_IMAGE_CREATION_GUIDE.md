## Quick Creation Methods

### Method 1: Using Canva (Recommended)
1. Go to [Canva.com](https://www.canva.com)
2. Create custom size: 1200 x 630 px
3. Choose "Social Media" template
4. Apply gradient background
5. Add text elements as specified
6. Add icons from Canva library
7. Download as JPG

### Method 2: Using Figma
1. Create new frame: 1200 x 630 px
2. Add gradient background
3. Add text layers
4. Import or create icons
5. Export as JPG (quality: 80%)

### Method 3: Using Photoshop
1. New document: 1200 x 630 px, 72 DPI
2. Create gradient layer
3. Add text with specified fonts
4. Add icon elements
5. Save for Web (JPG, quality: 80%)

### Method 4: Using Online Tools
- [Crello](https://crello.com)
- [Adobe Express](https://www.adobe.com/express/)
- [Snappa](https://snappa.com)
- [Visme](https://www.visme.co)

## Testing Your Image

### After Creating the Image:

1. **Save the file**:
   - Location: `c:\xampp\htdocs\Acmeerp-Website\acme_erp\public\og-image.jpg`
   - Ensure filename is exactly `og-image.jpg`

2. **Test on Social Media Debuggers**:
   
   **Facebook Debugger**:
   - Go to: https://developers.facebook.com/tools/debug/
   - Enter: `https://acmeerp.org`
   - Click "Scrape Again" to refresh cache
   
   **Twitter Card Validator**:
   - Go to: https://cards-dev.twitter.com/validator
   - Enter: `https://acmeerp.org`
   - Check preview
   
   **LinkedIn Post Inspector**:
   - Go to: https://www.linkedin.com/post-inspector/
   - Enter: `https://acmeerp.org`
   - Inspect

3. **Verify Image Loads**:
   - Open: `https://acmeerp.org/og-image.jpg`
   - Should display your created image
